# Perfil del usuario

## Rol y contexto
- Trabaja en o para Sustain Awards (sustainawards.com), empresa de trofeos personalizados
- Gestiona el desarrollo del "Agente Diseño Gráfico" — pipeline backend de IA para personalización de trofeos
- Perfil no puramente técnico: entiende el proyecto a nivel funcional, toma decisiones de producto, pero no escribe el código él mismo
- Trabaja en Windows 11, Python con venv

## Preferencias de comunicación
- Prefiere respuestas en español
- Le gustan las explicaciones claras y sencillas, sin demasiada jerga técnica innecesaria
- Aprecia los resúmenes con estructura visual (tablas, listas, títulos)
- Hace preguntas de negocio/producto antes de avanzar técnicamente — le gusta entender el "por qué" antes del "cómo"

## Estilo de trabajo
- Hace preguntas de validación antes de ejecutar cambios grandes ("antes quiero saber...")
- Pide contexto y verificación de que el estado del proyecto es correcto al retomar sesiones
- Le interesa el coste operativo del sistema (preguntó sobre costes de API)
- Piensa en el producto final desde la perspectiva del cliente de la empresa

## Notas de colaboración
- Al inicio de cada sesión sobre este proyecto, verificar si ya tienen la API key de Anthropic
- Cuando retome el proyecto, leer project_pending.md para saber el estado actual
- Explicar cambios técnicos en términos de impacto en el producto, no solo en el código
